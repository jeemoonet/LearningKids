# Word Hunter（单词猎人）

网页背单词战斗游戏 MVP。

## 运行

```bash
npm install
npm run dev
```

浏览器打开 http://localhost:5173

## 测试

```bash
npm test
```

## 构建

```bash
npm run build
```
