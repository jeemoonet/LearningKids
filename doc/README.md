## 项目简介

LearningKids 少儿互动单词学习 Web 应用，已重构为**以学员为中心的单词学习闭环**：

| 概念 | 说明 |
|------|------|
| 标准单词库 | 全平台共享的单词母库（初级/中级/高级） |
| 学习单词库 | 管理员在标准库基础上自定义的学习目标库 |
| 我的单词库 | 学员已掌握的单词（初始化自评 + 测评纳入） |
| 学习集 / 小节 | 每次抽 100 个未掌握词，拆成 5–10 个小节学习 |
| 学习课件 | 闪卡 / 选择题 / 拼写 / 完形 / 介词·句型·Word Hunter·**征服星球** 扩展游戏 |
| 测评 | 完形填空验收，通过后单词纳入我的库 |

核心约束：课件中例句、关联词、测试题用词只取自「我的单词库 ∪ 当前小节」。
多用户注册访问、学习数据隔离、标准/学习库全局共享。

技术栈：React 19 + TypeScript + Vite 8 + Hono + SQLite（node:sqlite，无 ORM）。
