## 项目简介

初中数学互动练习 Web 应用，当前包含：

| 模块 | 说明 |
|------|------|
| 函数图像 | 输入一次函数、二次函数或反比例函数，在同一坐标系中观察图像 |
| 正负训练营 | 闪卡练习正负号变换：去括号、变号与系数分配 |

技术栈：React 19 + TypeScript + Vite 8 + mathjs。
