# DOC-PLAN-001 项目计划

## 1. 项目概述

- **项目名称**：LearningKids
- **项目目标**：为少儿提供可交互的学习工具，覆盖数学练习与英语单词记忆
- **目标用户**：中小学生及其家长、教师
- **项目类型**：个人学习项目

## 2. 当前功能（MVP）

### 2.1 函数图像模块

- 支持一次函数、二次函数、反比例函数
- 最多同时绘制 3 条函数曲线
- 坐标系自适应缩放
- 内置测验模式（10 题）

### 2.2 正负训练营模块

- 闪卡练习：去括号、变号、系数分配
- 基础测试与进阶测试两种模式
- 限时答题
